URL Shortener Microservice

I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://codeinblack.com/node/api/tinyurl/new/https://www.google.com
https://codeinblack.com/node/api/tinyurl/new/https://bugraaydingoz.com

Example creation output:

{ "original_url":"https://bugraaydingoz.com", "short_url":"https://codeinblack.com/node/api/tinyurl/8170" }

Usage:

https://codeinblack.com/node/api/tinyurl/8170

Will redirect to:

https://bugraaydingoz.com
